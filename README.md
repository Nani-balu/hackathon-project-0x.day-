Wiring Connections
LoRa Module to Arduino (Transmitter)
LoRa Module Pins → Arduino Pins
-VCC → 5V (on Arduino)
-GND → GND (on Arduino)
-SCK → Pin 13 (on Arduino)
-MISO → Pin 12 (on Arduino)
-MOSI → Pin 11 (on Arduino)
-NSS → Pin 10 (on Arduino)
-RESET → Pin 9 (on Arduino)
-DIO0 → Pin 2 (on Arduino)
